# Stripe — Brand-saturated, gradient-confident

Vivid product hero, generous whitespace, real product visuals (not generic illustrations). Sans display + sans body in the same family. Use sparingly for fintech, payments, billing, and similar product marketing.

## Voice
- Plain English, no jargon.
- Numbers are first-class — "Pay $0 setup, $1.4 + 2.9% per charge."
- Each section reads like a press headline, not a feature list.

## Posture
- One vivid product gradient per page (hero only), never on cards.
- Borders are restrained; rely on whitespace to separate.
- Code samples for the developer audience, in mono with syntax highlighting.
