# Vercel — Mono-precise, dev-tools-grade

Minimal contrast palette, mono numerics, geometric sans. Layouts breathe at all sizes. Suits docs, dev consoles, and developer-marketing pages.

## Voice
- Technical, confident, no hedging.
- Code samples are first-class — show them, don't describe them.

## Posture
- High-contrast black on near-white; one accent only on interactive states.
- Hairline borders, no fills inside cards.
- Code blocks dark-on-light with mono font and copy buttons.
- Hover states subtle (background lift, no transforms).
