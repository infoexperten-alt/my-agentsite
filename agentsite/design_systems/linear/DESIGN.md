# Linear — Quiet, precise, software-native

A design system for product surfaces. Quiet chrome, content-led layouts, and a single decisive blue accent. Built for tools where the user is doing real work.

## Voice
- Precise, never marketing-fluffy.
- Verbs over adjectives. "Ship" not "world-class".
- Mono numerics for IDs, counts, durations.

## Posture
- Hairline borders only. No drop shadows except dropdowns/modals.
- Tight letter-spacing on display sizes (-0.02em).
- Sticky frosted nav, content-led layouts.
- One accent color (cobalt-ish) used at most twice per screen.
- Tabular nums on every metric.

## Inspirations
Linear, Notion 2024, Stripe docs, Vercel.
