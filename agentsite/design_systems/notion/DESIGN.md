# Notion — Approachable, paper-textured

Warm neutral canvas, friendly mono accents, generous reading column. Best for docs, notes, knowledge bases, and approachable product surfaces. Avoid for dense data tooling — use Linear instead.

## Voice
- Conversational but precise.
- Inline emoji used sparingly as section markers (📝, 💡, 🗂️) — not as feature icons.

## Posture
- Off-white background with a slight warm tint.
- Hairline borders, generous radii (8-12px).
- Inline page mentions formatted as pills with the page icon.
- Toggle-able callouts use a 1px accent border + tinted background.
